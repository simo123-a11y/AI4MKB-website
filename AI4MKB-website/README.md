# AI4MKB – Demo website

Een kleine voorbeeldwebsite voor AI4MKB, gebouwd met Python (Flask).

## Vereisten

- Python 3.9 of hoger
- `pip` om packages te installeren

## Installatie

In de map van dit project:

```bash
pip install -r requirements.txt
```

## Server starten

```bash
python app.py
```

Open daarna in je browser:

- `http://127.0.0.1:5000/`

## Structuur

- `app.py` – hoofdapplicatie (Flask)
- `templates/` – HTML-templates (`base.html`, `index.html`)
- `static/styles.css` – styling in AI/tech-look

Je kunt teksten, kleuren en secties naar wens aanpassen om nog beter aan te sluiten bij de uiteindelijke branding van AI4MKB.
